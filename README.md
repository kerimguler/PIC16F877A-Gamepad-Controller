# Gamepad Controller System

This project was developed using the PIC16F877A microcontroller to create a custom gamepad controller system. The system reads joystick, trigger, button, and MPU6050 sensor data and transmits the information to a computer through UART communication.

A Python application receives the incoming data and converts it into a virtual Xbox controller using the vgamepad library. The project also supports vibration feedback through a motor control circuit.

## Features

- Dual analog joystick support
- Analog trigger inputs
- Push button inputs
- MPU6050 accelerometer and gyroscope integration
- UART communication
- I2C communication
- Virtual Xbox controller support
- Vibration motor feedback
- Proteus simulation
- Hardware implementation

## Hardware Components

- PIC16F877A Microcontroller
- MPU6050 Accelerometer/Gyroscope Module
- HW-504 (KY-023) Analog Joystick Modules
- Push Buttons
- Vibration Motor
- PC817 Optocoupler
- LM324 Operational Amplifier
- Resistors and Capacitors

## System Overview

The PIC16F877A reads joystick positions, trigger values, button states, and MPU6050 sensor data. The collected information is transmitted to a computer through UART communication.

The Python application receives this data and maps it to a virtual Xbox controller using the vgamepad library. Force feedback generated by the virtual controller is transmitted back to the microcontroller, allowing vibration motor control.
